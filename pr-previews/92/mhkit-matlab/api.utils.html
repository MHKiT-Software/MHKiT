Utils ModuleÔÉÅ

The utils module contains the essential utility functions for data conversion, statistical analysis, caching, and event detection for the MHKiT library.

Note

The names of the functions below are of the convention path.path.function. Only the function name is used when calling the function in MATLAB. For example, to call on mhkit.wave.io.read_NDBC_file simply use read_NDBC_file.

Functions

Description

Calculate mean, max, min and stdev statistics of continuous data for a given statistical window.

Convert Excel datenum format to Python datetime.

Calculates magnitude and phase in two or three dimensions of the supplied vector.

bplot

This function will create a nice boxplot from a set of data.

mhkit.utils.check_name(input_name)ÔÉÅ
Check if a given string is a valid

field name for MATLAB struct.

If not, convert it to a valid field name

‚ÄúField names can contain ASCII letters (A‚ÄìZ, a‚Äìz), digits (0‚Äì9), and underscores, and must begin with a letter. The maximum length of a field name is $namelengthmax$.‚Äù ‚Äì MATLAB ref for struct

Parameters:

input_name (string) ‚Äì Name to check.

Returns:

  • output_name (the same as input_name if it is valid,) ‚Äì otherwise, it is the modified name.

  • 1. All special characters are replaced by ‚Äò_SC‚Äô

  • 2. Names not beigining with letters now begins with ‚ÄòA‚Äô.

  • 3. Name length exceeding $namelengthmax$ (=63) are truncated ‚Äì to name with a length of $namelengthmax$.

mhkit.utils.get_statistics(data, freq, options)ÔÉÅ

Calculate mean, max, min and stdev statistics of continuous data for a given statistical window. Default length of statistical window (period) is based on IEC TS 62600-3:2020 ED1. Also allows calculation of statistics for multiple statistical windows of continuous data.

Parameters:

  • data (strucutre) ‚Äì structure of variables to get statistics for with field called time.

  • freq (double or int) ‚Äì Sample rate of data [Hz]

  • period (double/int (optional)) ‚Äì Statistical window of interest [sec], default = 600 to call: get_statistics(data,freq,‚Äùperiod‚Äù,period)

  • vector_channels (cell array of strings (optional)) ‚Äì List of channel names that are to be vector averaged to call: get_statistics(data,freq,‚Äùvector_channels‚Äù,vector_channels)

Returns:

stats (structure) ‚Äì Structure with mean, max, min, and stdev of each variable
