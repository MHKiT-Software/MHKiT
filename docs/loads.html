Loads Module

The loads module contains a set of functions to calculate quantities of interest for assessing mechanical loads including statistics damage equivalent loads, and functions for visualizing data.

API Documentation

  • Python Loads API Documentation

  • MATLAB Loads API Documentation

Examples

  • Python Loads Example Notebook

  • MATLAB Loads Example LiveScript

    Calculations are based on IEC TS 62600-3:2020 ED1 .

The loads module contains the following submodules:

  • general: Computes loads metrics such as damage equivalent loads.

  • graphics: Generates graphics of loads statistics.

  • extreme: Contains functions to calculate peak distribution.

See MHKiT-Python or MHKiT-MATLAB for more details on the loads module.
